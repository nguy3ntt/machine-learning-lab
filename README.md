# Machine Learning Lab

This repository is my personal machine learning experiment lab. It contains small, focused notebooks exploring different machine learning models, methods, and concepts through practical implementation.

The goal of this repo to turn machine learning ideas into hands-on practice. Each notebook is designed to help me understand one specific topic more deeply by coding it, visualising it, testing it, and writing down the key takeaways.

---

## Purpose

Machine learning concepts can often stay too abstract when they are only learned through lectures, textbooks, or theory notes. This repository helps bridge that gap by turning those concepts into small experiments.

Each notebook focuses on questions like:

- What problem does this method solve?
- How does the model work at a high level?
- What does the model output actually mean?
- When does this method perform well?
- What are its limitations?
- How is it different from related methods?

The main purpose is learning, progression, and practice.

---

## Learning Approach

Each notebook is intentionally small and focused, each experiment usually covers one core idea.

A typical notebook may include:

1. A short explanation of the concept
2. A small dataset or synthetic example
3. Quick preprocessing and EDA
3. Model implementation
4. Basic visualisation
5. Simple evaluation or comparison
6. Key takeaways

This keeps the repo easy to maintain and allows each notebook to act as a clear learning record.
